Detailed Description

template<class Scalar, class FieldType>
class DataTransferKit::STKMeshFieldVector< Scalar, FieldType >

A class for defining Tpetra vectors over STK fields.

Use this class to manage Tpetra vectors and STK fields. This class will create a vector over a mesh set and the field on that mesh set. There is no gaurantee of consistency between the field and the vector as the vector does not point to the data in the field. Instead, the push and pull functions allow the user to move data between the vector and the field as necessary.
